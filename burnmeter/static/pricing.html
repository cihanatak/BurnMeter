Pro sync'i şimdi dene (self-host)
Billing henüz canlı değil; ama sync mekanizması çalışıyor ve self-host edilebilir — kendi relay'ini çalıştır:
pip install "burnmeter[sync]" # E2E için cryptography burnmeter sync-relay --port 8899 # relay (VPS / Tailscale node) — yalnızca ciphertext saklar # her makinede (AYNI passphrase): burnmeter sync login --relay http://<relay>:8899 --token <ortak-token> burnmeter sync push # bu cihazın özetini şifreleyip gönder
Sonra dashboard'daki Cihazlar kartında tüm makinelerini görürsün. Detay docs/SYNC.md + PRICING.md.
