Try Pro sync today (self-host)
Billing isn't live yet — but the sync engine works and is self-hostable. Run your own relay:
pip install "burnmeter[sync]" # cryptography for E2E burnmeter sync-relay --port 8899 # relay (VPS / Tailscale node) — stores ciphertext only # on every machine (SAME passphrase): burnmeter sync login --relay http://<relay>:8899 --token <shared-token> burnmeter sync push # encrypt & push this device's summary
Your machines then appear in the dashboard's Devices card. Details in docs/SYNC.md + PRICING.md.
