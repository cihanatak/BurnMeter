# Naming

**Burnmeter** is the name — final.

Vendor-neutral: the tool tracks **Claude Code + OpenAI Codex** today (more agents
planned), so the original Claude-only `cc`-prefixed name no longer fit. Short,
pronounceable, and it maps straight to the product's hero metric — your **burn rate**.

`burnmeter.dev` and the PyPI name `burnmeter` are secured.

## Considered & rejected
- **ccmeter** — the original name; vendor-locked to Claude Code (`cc`) and now misleading
  once Codex (and more) are tracked.
- **FuelGauge · BurnRate · Tollgate · Gauge** — taken or collide with existing tools/competitors.
- **Driftline · Headroom · Tideline** — clean availability, but abstract: they don't say what the tool does.
